Projects

World Happiness Dashboard

This interactive dashboard examines the relationships between World Happiness Index rankings and key factors such as population, education spending (as a percentage of GDP), and migration trends. By visualizing data, it highlights potential correlations and insights into the factors contributing to a nation's overall happiness.

Pet Image Classification Model

This project is an AI-powered tool that identifies different types of pets—like cats, dogs, birds, fish, hamsters, and bearded dragons—from uploaded photos. Trained on thousands of pet images, the model is hosted online, allowing users to easily upload a photo and see the pet type identified.

Screen Time Analysis Dashboard

This Excel dashboard analyzes screen usage patterns across age groups, focusing on middle-aged adults. It compares their usage with that of young and older adults to assess the potential impact on middle-aged individuals amidst their busy lives.

Tableau Portfolio

My Tableau Portfolio features interactive dashboards that turn complex data into actionable insights, covering areas like sales, customer segmentation, and performance tracking. Each project highlights my skills in data storytelling and visual analytics to support informed decision-making.
